# Hospital ER Analytics Dashboard

## Project Overview
Built 4-page Power BI Dashboard analyzing 
10,000+ patient records

## Tools Used
- Power BI
- DAX (12 Measures)
- Python (Dataset Creation)
- MS Excel

## Dashboard Pages
1. Overview - KPIs + Heatmap
2. SLA Performance - Breach Analysis  
3. Doctor Load - Performance Tracking
4. Root Cause - Readmission Analysis

## Key Metrics
- SLA Breach Rate: 48.7%
- Bed Occupancy: 76.49%
- Readmission Rate: 8.09%
- Total Patients: 10,000
